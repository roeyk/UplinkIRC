# Keyboard Shortcuts

---

## Message input

| Shortcut | Action |
|---|---|
| **Enter** | Send message |
| **Up Arrow** | Cycle back through previously sent messages |
| **Down Arrow** | Cycle forward through message history |
| **Tab** | Nick completion at cursor — press repeatedly to cycle through matches. Also completes slash commands (e.g. `/pi` → `/ping`). |

---

## Interface

| Shortcut | Action |
|---|---|
| **Ctrl+W** / window close button | Minimize to system tray (does not quit) |
| **Ctrl+F** | Open the message search bar in the current buffer |
| **Escape** (in search bar) | Close search and clear the highlight |
| **Escape** (with reply pending) | Cancel the pending reply |

---

## Search bar (Ctrl+F)

| Shortcut | Action |
|---|---|
| **Enter** | Jump to the next match (wraps around) |
| **Shift+Enter** | Jump to the previous match |
| **Escape** | Close the search bar |
