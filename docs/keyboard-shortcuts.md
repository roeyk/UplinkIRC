# Keyboard Shortcuts

---

## Message input

| Shortcut | Action |
|---|---|
| **Enter** | Send message |
| **Up Arrow** | Cycle back through previously sent messages *(planned)* |
| **Down Arrow** | Cycle forward through message history *(planned)* |
| **Tab** | Nick completion at cursor *(planned)* |

---

## Interface

| Shortcut | Action |
|---|---|
| **Ctrl+W** / window close button | Minimize to system tray (does not quit) |

---

## Planned

The following shortcuts are on the roadmap and will be added in future releases:

- **Tab** — complete nicks in the current channel, cycling through matches
- **Up / Down arrows** — recall previously sent messages
- **Alt+Up / Alt+Down** — navigate between channels in the sidebar
- **Ctrl+F** — search messages in the current buffer
