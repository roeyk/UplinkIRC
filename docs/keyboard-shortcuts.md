# Keyboard Shortcuts

---

## Message input

| Shortcut | Action |
|---|---|
| **Enter** | Send message |
| **Up Arrow** | Cycle back through previously sent messages |
| **Down Arrow** | Cycle forward through message history |
| **Tab** | Nick completion at cursor — press repeatedly to cycle through matches |

---

## Interface

| Shortcut | Action |
|---|---|
| **Ctrl+W** / window close button | Minimize to system tray (does not quit) |

---

## Planned

The following shortcuts are on the roadmap and will be added in future releases:

- **Alt+Up / Alt+Down** — navigate between channels in the sidebar
- **Ctrl+F** — search messages in the current buffer
