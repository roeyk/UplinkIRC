Why Uplink

Built for long sessions

Designed to stay connected and stable for hours or days — a client named for what it does.

🔒

TLS by default

Every connection uses TLS via Qt's QSslSocket. Plain IRC is not an option — your connection is encrypted from the first byte.

⚡

Lightweight & native

Built with Qt6 Widgets and C++17. No web runtime, no Electron, no Chromium. Small binary, small RAM footprint.

📡

IRCv3 first

CAP LS 302 negotiation, message-tags, server-time, multi-prefix, away-notify, batch, and labeled-response — with more on the way.

🎨

55 built-in themes

Catppuccin, Nord, Dracula, Gruvbox, Solarized, Tokyo Night, and dozens more. Switch live from the hamburger menu.

🖥

System tray

Close the window and keep running in the tray. Unread badge, balloon notifications, and one-click restore.

🔌

Bouncer-ready

Connect through ZNC or soju using the password field. PASS is sent before registration — just how bouncers expect it.

🔑

SASL & NickServ

SASL PLAIN authenticates your nick before you appear on the network. For older servers, nickserv_password sends IDENTIFY automatically on connect.

📐

Persistent layout

Sidebar size and position are saved when you quit and restored on next launch. Drag it to exactly where you want — it stays there.

📋

Nick list & modes

Full prefix-rank sorting (~&@%+). Live updates on JOIN, PART, QUIT, NICK, KICK, and MODE. Click ⚙ to collapse or expand the panel. Topic bar shows channel modes.

🛠

TOML config

One readable config file. Multiple servers, channels on one line (channels = "#uplink, #linux"), UI toggles. Auto-created on first launch with sensible defaults.

🧱

Stable memory

Message buffer capped at 500 per channel. RAM usage stays flat across long sessions on busy networks.

📁

DCC file transfer

Send files directly to any user via DCC. Both active and passive modes supported — passive DCC lets you send files even when you're behind a home router or NAT.

🔄

In-app update check

Click ☰ → Check for Updates. Uplink queries the GitHub releases API and tells you whether a newer version is available — one click, no browser needed.

IRCv3

Modern IRC, fully negotiated

CAP LS 302 — Uplink waits for the server's full capability list before requesting anything, so it never NAKs on unsupported caps.
